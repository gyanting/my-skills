# 时间关联判定规则（基于 JSON Key）

## 输入说明

- 每个厂商的 JSON 文件包含多个 Key，每个 Key 代表一个时间窗口
- Key 格式：`开始时间_结束时间`
- 示例：`2025-01-01 10:00:00_2025-01-01 11:00:00`
- 时间分辨率：秒

---

## 关联（is_related = true）

| 关联类型 | 判定条件 | 示例（基于 Key） |
|----------|----------|------------------|
| **完全包含** | 厂商A的某个时间窗口完全包含厂商B的某个时间窗口 | A：`10:00:00_11:00:00`；B：`10:30:00_10:45:00` |
| **部分重叠** | 两个时间窗口有交集，但不完全包含 | A：`10:00:00_11:00:00`；B：`10:30:00_11:30:00` |
| **首尾相接** | 两个时间窗口精确衔接，间隔 = 0 秒 | A：`10:00:00_11:00:00`；B：`11:00:00_12:00:00` |

**为什么这三种都算关联：** 都意味着两台设备很可能见到了同一批网络流量。差异源自检测能力差异，而非数据来源不同——这是后续四元组对比的前提。

---

## 不关联（is_related = false）

| 不关联类型 | 判定条件 | 示例（基于 Key） |
|------------|----------|------------------|
| **时间断层** | 两个时间窗口间隔在 1 秒到 300 秒（5分钟）之间 | A：`10:00:00_11:00:00`；B：`11:00:05_12:00:00`（间隔5秒） |
| **完全分离** | 两个时间窗口间隔 > 300 秒（5分钟） | A：`10:00:00_11:00:00`；B：`14:00:00_15:00:00`（间隔3小时） |

**注**：若两个厂商各有多个时间窗口，需逐对判断。只要存在任意一对满足「完全包含/部分重叠/首尾相接」，即判定为关联。

不关联时不应做四元组对比——双方看到的本就不是同一段流量，差异不再有"检测能力"的含义。这时改走独立分析路径，各自分析即可。

---

## 配对原则

| 原则 | 说明 |
|------|------|
| **一对一** | 任一 JSON 文件最多与一个厂商的 JSON 文件配对。原因：让"独有告警数"严格等于"对方未检测到的告警数"，否则同一份流量被多对重复计入，差异统计会虚高。 |
| **优先跨厂商配对** | 竞品对比（如绿盟 vs 微步）比同厂商对比信息量大得多。分两轮：先跨厂商配对，剩余文件再同厂商配对。 |
| **关联类型优先级** | 完全包含 (3) > 部分重叠 (2) > 首尾相接 (1)。当多个配对候选时，优先选择优先级高的。 |

---

## 多个时间窗口的处理规则

当一个厂商的 JSON 包含多个 Key（多个时间窗口）时：

| 场景 | 判定规则 |
|------|----------|
| **A有3个窗口，B有1个窗口** | 只要 B 的窗口与 A 的**任意一个**窗口满足关联条件，即判定为关联 |
| **A有3个窗口，B有2个窗口** | 逐对检查，存在任意一对满足关联条件即判定为关联 |
| **部分窗口关联、部分不关联** | 整体判定为关联，但报告需标注：仅对关联窗口进行四元组对比，非关联窗口单独分析 |

---

## 示例说明

### 示例1：完全包含（关联）
- A窗口：2025-01-01 10:00:00_2025-01-01 11:00:00
- B窗口：2025-01-01 10:30:00_2025-01-01 10:45:00
- 判定：关联（完全包含）
- 建议：五元组对比分析


### 示例2：部分重叠（关联）
- A窗口：2025-01-01 10:00:00_2025-01-01 11:00:00
- B窗口：2025-01-01 10:30:00_2025-01-01 11:30:00
- 判定：关联（部分重叠）
- 建议：五元组对比分析


### 示例3：首尾相接（无关联）
- A窗口：2025-01-01 10:00:00_2025-01-01 11:00:00
- B窗口：2025-01-01 11:00:00_2025-01-01 12:00:00
- 判定：无关联（首尾相接）
- 建议：独立分析


### 示例4：时间断层（不关联）
- A窗口：2025-01-01 10:00:00_2025-01-01 11:00:00
- B窗口：2025-01-01 11:00:05_2025-01-01 12:00:00
- 判定：不关联（时间断层，间隔5秒）
- 建议：独立分析，不进行五元组对比-
